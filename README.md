📌 1. Project Title
AI-Powered Text to Speech Converter

📌 2. Project Overview

## Project Overview
This project is a web-based Text to Speech application that converts written text into spoken audio using AI speech synthesis. It also supports PDF text extraction, voice selection, and audio download features.

📌 3. Features

## Features
- Authentication system
- PDF processing
- Text to speech conversion
- Male/Female voice selection
- Speed control
- Audio download
- Speech history
- Responsive UI
